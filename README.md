# smart-library-management-system
Final year project using ML + Streamlit + MySQL

## Description 
A smart digital library solution built using Python, Streamlit, MySQL and Machine Learning. The system digitalizes book issuing,returning,tracking and uses ML for smart predictions and recommendations.

## Features
• Student & Book Management
• Issue / Return Books
• Fine Calculation
• Streamlit Web UI
• ML Modules included
  -->Book Recommendation
  -->Due-Date Prediction
  -->Most Demanded Book Prediction
  -->Book Demand Forecasting
